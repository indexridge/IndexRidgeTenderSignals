# Marketplace comparison: Gumroad vs Payhip

Objective summary for IndexRidge Tender Signals / Public Data Packs.

## Bottom line

- **Best if you need marketplace/discovery:** Gumroad. It has Gumroad search/Discover and is closer to a marketplace, but fees are higher and organic discovery is not guaranteed.
- **Best if we drive traffic ourselves from GitHub Pages/SEO/email/ads:** Payhip. It is usually lower-fee on the free plan, but it is more of a storefront/checkout than a marketplace.

Plain summary: Best if you need marketplace/discovery: Gumroad. Best if we drive traffic ourselves from GitHub Pages/SEO/email/ads: Payhip.

For this project, start with **Gumroad first** because the user asked for Gumroad and because marketplace/discovery is more valuable before we have our own traffic. Add Payhip later as a lower-fee direct checkout if the public pages start getting visits.

## Fee and discovery notes to re-check before launch

Public pricing/search results checked 2026-06-29 indicate:

- Gumroad: public pricing page advertises no monthly fee and a simple 10% cut per sale. Some third-party comparisons mention additional payment-processing/effective-fee considerations and higher fees for marketplace/discover-originated sales; verify on Gumroad before publishing.
- Payhip: public pricing/search results indicate a free plan with transaction fee and paid plans that reduce platform fee. Verify current Payhip pricing before publishing.
- Gumroad has a buyer-facing marketplace/discovery surface. Payhip generally requires you to bring your own traffic.

## Platform fit for these products

| Criterion | Gumroad | Payhip |
| --- | --- | --- |
| Marketplace/discovery | Better fit | Weaker fit |
| Direct checkout fees | Usually higher | Usually lower on comparable direct sales |
| Setup speed | Very fast | Fast |
| Digital downloads | Good | Good |
| Subscriptions/memberships | Available | Available |
| Customer tax/VAT handling | Verify in seller settings | Verify in seller settings |
| Best use here | First public paid listing | Lower-fee backup/direct checkout |

## No-bias recommendation

1. Use **Gumroad first** for a low-price data-pack listing because it is closest to a marketplace and can be launched quickly.
2. Use **Payhip second** if we are mainly linking from our own GitHub Pages sites or if Gumroad fees materially reduce margin.
3. Do not assume either platform creates demand by itself. The products are live technically, but revenue still depends on buyers finding the listing and trusting the offer.
